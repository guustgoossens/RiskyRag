# RiskyRag UI Mockup Previews

This folder contains prompts and generated images for UI concept mockups.

## Files

Each prompt file (`.txt`) corresponds to a specific UI concept. Generated images should be saved with matching names:

- `01-game-board.txt` → `01-game-board.png`
- `02-ai-negotiation.txt` → `02-ai-negotiation.png`
- `03-tool-usage-dashboard.txt` → `03-tool-usage-dashboard.png`
- `04-historical-query.txt` → `04-historical-query.png`
- `05-scenario-selection.txt` → `05-scenario-selection.png`
- `06-benchmark-results.txt` → `06-benchmark-results.png`

## Usage

1. Copy prompts from `.txt` files
2. Generate images using Nano Banana Pro or similar AI image generator
3. Save generated images in this folder with matching filenames
4. Use images for pitch deck, README, and demo materials

## Purpose

These mockups showcase RiskyRag's core features:
- Temporal RAG filtering (knowledge cutoff enforcement)
- LLM agent strategic gameplay
- Educational historical queries
- Real-time tool usage monitoring
- Comprehensive benchmarking
